## Welcome

VideoAiditor offers a powerful toolkit to streamline your video editing workflows.

## Authentication

All API endpoints are authenticated using API keys. Include your API key in the request header as `videoaiditor_api_key`.

## Supported languages

We currently support english language. We plan to add support for other languages in the near future.
